Frequency, in GHz.

lumos.model.workload.build(app_num, kernels, cov_dist, cov_param1, cov_param2)[source]¶

Build a workload composed of applications with the specified kernels coverage following a specified distribution.

Args:
app_num (int):
The number of applications in a workload.
kernels (list):
The list of kernel IDs.
cov_dist (str):
The distribution type of coverage, currently supports normal, lognormal, uniform.
cov_param1,cov_param2 (number):
The statistical parameters for the distribution. Their real meanings depend on the specific distribution. For normal and lognormal, param1 is the mean, and param2 is the standard deviation. For uniform, param1 is the minimum range, and param2 is the maximum range.
Returns:
workload (list):
A list of successfully built applications.
lumos.model.workload.build_fixedcov(app_num, kernels, cov)[source]¶

Build a workload composed of applications each of which includes a kernel with the given coverage. The selection of a kernel for an application follows the occur probability of that kernel.

Args:
app_num (int):
The number of applications in a workload.
kernels (list):
The list of kernel IDs.
cov (num):
The fixed coverage for each kernel once it presents in an application, e.g. 0.4
Returns:
workload (list):
A list of successfully built applications.
